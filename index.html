Hello there!

Welcome to the unofficial DHS notes website. In here, you will find Year 1 and Year 2 notes in Dunman High, written by various people.

Please do take a look around!

If you are a student currently studying in Dunman High, feel free to use these notes as your reference when studying. however, if you are not a Dunman High student, please refrain from distributing and sharing these notes without permission, as these notes are only allowed to be internally distributed with the creator's permission.

If you have further inquiries, drop an email at dhschool.notes@gmail.com.


Github Repository

This website is part of a Github repository found here. If you would like to contribute to the notes or the website, head over to that repository and take a look at the README.md file.


Disclaimer

If you happen to get low marks on any exam while using notes found here for revision, keep your cool and be civilised.

These notes are not 100% guaranteed to get you high marks, so please proceed at your own risk.

If you feel like the notes found here are insufficient/inaccurate, please help others out by contributing to the repository or opening an issue with constructive, specific feedback.


Guidelines

Practice integrity, do not copy or plagiarise notes.

For internal circulation only.

Be kind.
